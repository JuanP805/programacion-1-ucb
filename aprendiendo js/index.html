Integra tus funciones con estilo profesional

Este proyecto base es responsivo, usa tus colores primario y secundario, e incluye un botón que ejecuta miPrimerAlgoritmo() desde tu archivo aprendiendo.js.

Presiona para ejecutar la función miPrimerAlgoritmo y ver el resultado debajo.

¿Cómo agregar tus funciones?

  1. Crea/edita aprendiendo.js en la misma carpeta.
  2. Define function miPrimerAlgoritmo() { ... } y retorna un valor o texto.
  3. Pulsa el botón para ver la salida aquí mismo.

Consejo: retorna un objeto o string; si es objeto/array, lo verás como JSON.
